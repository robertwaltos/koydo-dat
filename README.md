# koydo-dat

DAT exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM168 |
| Name | DAT |
| Country | United States/Canada |
| Category | Admissions |
| Tier | Tier 8 |
| Wave | Wave 10 |
